## 핵심 기능
- **`플레이리스트 추가` :** 구글 로그인 후 유튜브 재생목록 추가 구현
- **`소셜로그인 구현` :** Oauth 구글 로그인 구현
- **`영상 플레이리스트 추가` :** 모달 및 차트에서 영상 플레이리스트 추가 가능
- **`인기리스트` :** 실제 유튜브사이트 내 Top10 영상 확인기능
- **`영상재생 기능` :** 메인페이지 하단 유튜브 플레이어 기능으로 영상 구현
